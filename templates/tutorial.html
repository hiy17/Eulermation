What is an Eulerian Path? ▼

An Eulerian Path is a path in a graph that visits every edge exactly once. It may or may not visit every vertex, and it can start and end at different vertices.

  • Exists in a graph if it is connected and has exactly 0 or 2 vertices with odd degree.
  • If there are 0 odd-degree vertices, the path can form a circuit (see Eulerian Circuit).
  • If there are 2 odd-degree vertices, the path starts at one and ends at the other.
What is an Eulerian Circuit? ▼

An Eulerian Circuit is a special case of an Eulerian Path that is a closed loop, starting and ending at the same vertex, covering every edge exactly once.

  • Exists in a graph if it is connected and all vertices have even degree.
  • A graph with an Eulerian Circuit is called an Eulerian Graph.
Properties of Eulerian Graphs ▼

Understanding the properties of a graph is key to identifying Eulerian Paths and Circuits:

  • Connectedness: The graph must be connected (there exists a path between any pair of vertices).
  • Vertex Degree:
    • Eulerian Circuit: All vertices must have even degree.
    • Eulerian Path: Exactly 0 or 2 vertices have odd degree.
  • Directed Graphs: For directed graphs, each vertex must have equal in-degree and out-degree for a circuit, or at most one vertex with out-degree - in-degree = 1 and another with in-degree - out-degree = 1 for a path.
How to Find an Eulerian Path/Circuit ▼

To determine if a graph has an Eulerian Path or Circuit and find it, follow these steps:

  • Check Connectedness: Ensure the graph is connected using DFS or BFS.
  • Count Vertex Degrees:
    • If all vertices have even degree and the graph is connected, an Eulerian Circuit exists.
    • If exactly 2 vertices have odd degree, an Eulerian Path exists.
  • Use Fleury’s Algorithm:
    • Start at a vertex (preferably an odd-degree vertex for a path).
    • Traverse edges, ensuring not to cut the graph into two components unless the edge is the last option.
    • Remove edges as they are traversed and continue until all edges are covered.
    • Applications of Eulerian Paths ▼

    Eulerian Paths and Circuits have practical applications in various fields:

    • Graph Theory: Solving problems like the Seven Bridges of Königsberg.
    • Network Design: Optimizing routes for mail delivery, snow plowing, or garbage collection.
    • Bioinformatics: Assembling DNA sequences by finding Eulerian Paths in de Bruijn graphs.
    • Computer Science: Used in algorithms for graph traversal and circuit design.
    Checking if a Graph is Eulerian ▼

    To determine if a graph is Eulerian (has an Eulerian Path or Circuit), you need to verify specific conditions efficiently.

  • Verify Connectedness: Use Depth-First Search (DFS) to ensure all vertices with non-zero degree are connected.
  • Analyze Vertex Degrees:
    • Count vertices with odd degree. If more than 2, no Eulerian Path exists.
    • If exactly 0 odd-degree vertices, an Eulerian Circuit exists (if connected).
    • If exactly 2 odd-degree vertices, an Eulerian Path exists.

    • This algorithm runs in O(V + E) time, where V is the number of vertices and E is the number of edges. Eulirian Path & Circuit

    Hierholzer’s Algorithm for Eulerian Circuits ▼

    Hierholzer’s Algorithm is an efficient method to find an Eulerian Circuit in a graph with all vertices of even degree. It is faster than Fleury’s as it avoids checking for bridges.

  • Choose a Starting Vertex: Start at any vertex.
  • Follow Edges: Traverse unused edges, pushing vertices onto a stack, until no edges remain from the current vertex.
  • Backtrack: Pop vertices from the stack, adding them to the circuit, and explore remaining edges from each popped vertex.
  • Reverse the Circuit: The final circuit is obtained by reversing the collected vertices.

    • This algorithm is highly efficient with O(E) time complexity. Hierholzer’s Algorithm Directed Graph

    Euler’s Theorem ▼

    Euler’s Theorem gives a necessary and sufficient condition for the existence of an Eulerian Path or Circuit:

    • Eulerian Circuit: A graph has an Eulerian Circuit if and only if every vertex has an even degree, and the graph is connected.
    • Eulerian Path: A graph has an Eulerian Path if and only if exactly two vertices have odd degree, and the graph is connected.
