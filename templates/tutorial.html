What is an Eulerian Circuit? â–¼

An Eulerian Circuit is a special case of an Eulerian Path that is a closed loop, starting and ending at the same vertex, covering every edge exactly once.

  • Exists in a graph if it is connected and all vertices have even degree.
  • A graph with an Eulerian Circuit is called an Eulerian Graph.
Properties of Eulerian Graphs â–¼

Understanding the properties of a graph is key to identifying Eulerian Paths and Circuits:

  • Connectedness: The graph must be connected (there exists a path between any pair of vertices).
  • Vertex Degree:
    • Eulerian Circuit: All vertices must have even degree.
    • Eulerian Path: Exactly 0 or 2 vertices have odd degree.
  • Directed Graphs: For directed graphs, each vertex must have equal in-degree and out-degree for a circuit, or at most one vertex with out-degree - in-degree = 1 and another with in-degree - out-degree = 1 for a path.
How to Find an Eulerian Circuit â–¼

To find an Eulerian Circuit in a graph, follow these steps:

  • Check Connectedness: Make sure the graph is connected. You can do this using Depth-First Search (DFS) or Breadth-First Search (BFS).
  • Verify Vertex Degrees: Confirm that every vertex has an even degree. If any vertex has an odd degree, an Eulerian Circuit is not possible.
  • Apply Fleuryâ€™s Algorithm or use Hierholzerâ€™s Algorithm:
    • Start at any vertex.
    • Follow edges one at a time, avoiding bridges unless no other options are available.
    • Keep track of visited edges and continue until you return to the starting vertex having covered every edge exactly once.
    • Checking if a Graph is Eulerian â–¼

    To determine if a graph is Eulerian (has an Eulerian Path or Circuit), you need to verify specific conditions efficiently.

  • Verify Connectedness: Use Depth-First Search (DFS) to ensure all vertices with non-zero degree are connected.
  • Analyze Vertex Degrees:
    • Count vertices with odd degree. If more than 2, no Eulerian Path exists.
    • If exactly 0 odd-degree vertices, an Eulerian Circuit exists (if connected).
    • If exactly 2 odd-degree vertices, an Eulerian Path exists.

    • This algorithm runs in O(V + E) time, where V is the number of vertices and E is the number of edges.
